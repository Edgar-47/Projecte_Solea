# Projecte_Solea
## Solea – Projecte de Desenvolupament Web
### Edgar Pedret Gironès
#### Descripció del Projecte

Aquest projecte forma part d’una pràctica de classe i consisteix en el desenvolupament complet d’una pàgina web per a l’empresa de càtering Solea. El disseny inicial s’ha creat amb Figma, incloent una guia d’estils i una identitat visual pròpia per assegurar coherència en tot el lloc web.

#### Tecnologies i Estructura

- Implementació d’estructura HTML5 amb etiquetes semàntiques.

- Inclusió de metadades, microdata i Schema.org per millorar el SEO i l’accessibilitat.

- Validació completa del codi HTML i CSS amb les eines oficials del W3C i correcció dels avisos detectats.

- Creació del full d’estils CSS personalitzat.

- Integració de Bootstrap per agilitzar el desenvolupament i afegir components responsius.

#### Compliment Legal

S’han aplicat les normatives LOPD i RGPD mitjançant:

- Banner de cookies funcional.

- Pàgina de política de privadesa.

- Casella obligatòria d’acceptació al formulari per al tractament de dades.

#### Desplegament

El lloc web s’ha hostatjat a GitHub Pages, una solució eficient per a webs estàtiques que no requereixen configuracions complexes.

#### Responsive Design

S’ha comprovat el correcte funcionament del disseny en diversos dispositius i resolucions per garantir una experiència òptima en mòbils, tauletes i escriptoris.
